A2A (Agent-to-Agent) Protocol

A2A (智能体对智能体) 协议

A2A (エージェント対エージェント) プロトコル

The standard defining how autonomous AI agents discover, communicate, negotiate, and collaborate to solve complex problems without human intervention.

定义自主 AI 智能体如何发现、通信、协商和协作以在无需人工干预的情况下解决复杂问题的标准。

自律型AIエージェントが、人間の介入なしに複雑な問題を解決するために、どのように発見、通信、交渉、協力するかを定義する標準です。

Core Concepts

核心概念

コアコンセプト

Discovery

发现

発見

Agents publish their capabilities (e.g., "I can search the web" or "I can book flights") to a registry so other agents can find them.

智能体将其能力（例如“我可以搜索网络”或“我可以预订航班”）发布到注册表中，以便其他智能体可以找到它们。

エージェントは、自分の機能（例：「Web検索ができます」または「フライトを予約できます」）をレジストリに公開し、他のエージェントが見つけられるようにします。

Handshake & Auth

握手与认证

ハンドシェイクと認証

Establishing a secure channel. Verifying the identity of the agent (e.g., using DID or mTLS) and agreeing on a communication schema.

建立安全通道。验证智能体的身份（例如，使用 DID 或 mTLS）并就通信模式达成一致。

安全なチャネルの確立。エージェントのIDを検証し（例：DIDまたはmTLSを使用）、通信スキーマに合意します。

The Standards Landscape

标准格局

標準のランドスケープ

Protocol 协议 プロトコル Status 状态 ステータス Description 描述 説明
Agent Protocol (AP) Active / Emerging A modern open standard for agent communication, focusing on task exchange and artifact handling. 现代开放标准，专注于任务交换和工件处理。 タスク交換とアーティファクト処理に焦点を当てた、エージェント通信のための最新のオープン標準。
FIPA ACL Legacy / Academic The original Foundation for Intelligent Physical Agents standard. Highly formal, based on speech act theory. 最初的智能物理代理基础标准。高度形式化，基于言语行为理论。 インテリジェント物理エージェントのための基盤（FIPA）のオリジナル標準。スピーチアクト理論に基づいた非常に形式的なもの。
MCP (Model Context Protocol) Active / Tooling While primarily for User-Agent-Tool connection, it's increasingly used for Agent-to-Agent tool sharing. 虽然主要用于用户-代理-工具连接，但越来越多地用于智能体对智能体的工具共享。 主にユーザー-エージェント-ツールの接続に使用されますが、エージェント対エージェントのツール共有にもますます使用されています。
