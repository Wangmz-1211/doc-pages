Microsoft Prompt Flow

微软 Prompt Flow

Microsoft Prompt Flow

A suite of development tools designed to streamline the entire development cycle of AI applications powered by Large Language Models (LLMs).

一套旨在简化由大型语言模型 (LLM) 驱动的 AI 应用程序的整个开发周期的开发工具。

大規模言語モデル (LLM) を活用した AI アプリケーションの全開発サイクルを効率化するために設計された開発ツール群です。

What is Prompt Flow?

什么是 Prompt Flow？

Prompt Flow とは？

Prompt Flow is a comprehensive tool developed by Microsoft that enables developers to create, debug, evaluate, and deploy LLM-based applications. It focuses on connecting various models, prompts, and Python code into an executable flow. [1]

Prompt Flow 是微软开发的一款综合工具，使开发人员能够创建、调试、评估和部署基于 LLM 的应用程序。它专注于将各种模型、提示和 Python 代码连接成可执行的流。 [1]

Prompt Flow は Microsoft によって開発された包括的なツールであり、開発者が LLM ベースのアプリケーションを作成、デバッグ、評価、およびデプロイできるようにします。さまざまなモデル、プロンプト、Python コードを実行可能なフローに接続することに重点を置いています。 [1]

With Prompt Flow, developers can visualize their workflow as a Directed Acyclic Graph (DAG), making it easier to understand the dependencies and logic of complex AI interactions. It integrates seamlessly with Azure Machine Learning and VS Code.

使用 Prompt Flow，开发人员可以将他们的工作流程可视化为有向无环图 (DAG)，从而更容易理解复杂 AI 交互的依赖关系和逻辑。它与 Azure Machine Learning 和 VS Code 无缝集成。

Prompt Flow を使用すると、開発者はワークフローを有向非巡回グラフ (DAG) として視覚化でき、複雑な AI インタラクションの依存関係とロジックを理解しやすくなります。Azure Machine Learning および VS Code とシームレスに統合されます。

Key Capabilities

主要功能

主な機能

  • Flow Engineering: Create executable flows that link LLMs, prompts, and Python code.
  • 流工程： 创建链接 LLM、提示和 Python 代码的可执行流。
  • フローエンジニアリング： LLM、プロンプト、Python コードをリンクする実行可能なフローを作成します。
  • Debugging: Interactively debug flows with ease, tracing data through each node.
  • 调试： 轻松交互式调试流，通过每个节点跟踪数据。
  • デバッグ： フローを簡単に対話的にデバッグし、各ノードを通してデータをトレースします。
  • Evaluation: Calculate quality and performance metrics for your flows using larger datasets.
  • 评估： 使用更大的数据集计算流的质量和性能指标。
  • 評価： 大規模なデータセットを使用して、フローの品質とパフォーマンスメトリクスを計算します。
