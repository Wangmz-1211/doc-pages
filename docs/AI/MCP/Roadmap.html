Roadmap & Ecosystem 路线图与生态系统 ロードマップとエコシステム

Current Status 当前状态 現在のステータス

MCP is currently an open standard released under the Apache 2.0 license. It is stable for production use and is rapidly being adopted by major development tools.

MCP 目前是根据 Apache 2.0 许可证发布的开放标准。它在生产使用中是稳定的，并且正迅速被主要开发工具采用。

MCPは現在、Apache 2.0ライセンスの下で公開されているオープン標準です。本番環境での使用において安定しており、主要な開発ツールによって急速に採用されています。

TypeScript SDK

Official support for Node.js and browser environments. 官方支持 Node.js 和浏览器环境。 Node.jsおよびブラウザ環境の公式サポート。

npm install @modelcontextprotocol/sdk

Python SDK

Official support for synchronous and asynchronous Python. 官方支持同步和异步 Python。 同期および非同期Pythonの公式サポート。

pip install mcp

Future Roadmap 未来路线图 今後のロードマップ

  • Authentication: 身份验证: 認証: Standardizing stronger auth patterns for remote servers (beyond OAuth). 标准化远程服务器的更强身份验证模式（超越 OAuth）。 リモートサーバーのためのより強力な認証パターンの標準化（OAuth以外）。
  • Transport Layers: 传输层: トランスポート層: Expanding beyond stdio and SSE to support more networking protocols. 扩展标准输入/输出和 SSE 之外的支持，以支持更多的网络协议。 stdioやSSEを超えて、より多くのネットワークプロトコルをサポートするように拡張。
  • Community: 社区: コミュニティ: Growing the library of open-source MCP servers for popular services. 为流行服务扩展开源 MCP 服务器库。 人気のあるサービス向けのオープンソースMCPサーバーのライブラリを拡大。
