Democratizing Intelligence

智能民主化

知能の民主化

As the year closed, the "David vs. Goliath" narrative took center stage. DeepSeek R1 and Google's LiteRT proved that you don't need a billion-dollar data center to run world-class AI.

随着一年的结束，“大卫与歌利亚”的故事占据了中心舞台。DeepSeek R1 和 Google 的 LiteRT 证明，你不需要十亿美元的数据中心就能运行世界级的 AI。

年が明けるにつれて、「ダビデ対ゴリアテ」の物語が中心的な舞台となりました。DeepSeek R1とGoogleのLiteRTは、世界クラスのAIを実行するのに10億ドルのデータセンターは必要ないことを証明しました。

Local First

本地优先

ローカルファースト

The release of DeepSeek R1 (Apache 2.0 licensed) triggered a wave of innovation. Startups previously priced out of the API market began building custom, fine-tuned models on their own hardware.

DeepSeek R1（Apache 2.0 许可）的发布引发了一波创新浪潮。此前被 API 市场价格拒之门外的初创公司开始在自己的硬件上构建定制的、微调的模型。

DeepSeek R1（Apache 2.0ライセンス）のリリースは、イノベーションの波を引き起こしました。以前はAPI市場の価格設定から締め出されていたスタートアップ企業が、自社のハードウェア上でカスタムのファインチューニングされたモデルを構築し始めました。

LiteRT on Android

Android 上的 LiteRT

Android上のLiteRT

Google demonstrated a real-time translation app running entirely on a mid-range Pixel phone. It handled 15 languages simultaneously with zero server latency, preserving user privacy completely. Google 展示了一款完全在中端 Pixel 手机上运行的实时翻译应用程序。它可以同时处理 15 种语言，零服务器延迟，完全保护用户隐私。 Googleは、ミッドレンジのPixelスマートフォン上で完全に動作するリアルタイム翻訳アプリを実演しました。15の言語を同時に処理し、サーバーの遅延はゼロで、ユーザーのプライバシーを完全に保護しました。
