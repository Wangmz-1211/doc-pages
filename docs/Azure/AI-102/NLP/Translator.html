1. Custom Translator

When standard translation lacks industry-specific terminology (e.g., Legal, Medical, Automotive), use Custom Translator to build a NMT model.

当标准翻译缺乏特定行业的术语（例如，法律、医疗、汽车）时，使用自定义翻译来构建 NMT 模型。

2. No-Trace Policy

2. 无痕迹策略

2. No-Trace ポリシー

Unlike some other services, Azure Translator is No-Trace. Submitted text is processed in memory and never written to persistent storage, ensuring data privacy for enterprise use cases.

与其他一些服务不同，Azure 翻译是无痕迹的。提交的文本在内存中处理，从不写入持久存储，确保企业用例的数据隐私。

他のいくつかのサービスとは異なり、Azure Translator は No-Trace です。送信されたテキストはメモリ内で処理され、永続ストレージに書き込まれることはないため、エンタープライズユースケースのデータプライバシーが保証されます。
