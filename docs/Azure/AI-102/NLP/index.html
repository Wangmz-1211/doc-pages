Exam Strategy

考试策略

試験戦略

  • Know the Limits: Memorize tier limits (Free vs Standard) and character counts. 了解限制： 记住层级限制（免费与标准）和字符数。 制限を知る： ティア制限（無料 vs 標準）と文字数を暗記します。
  • Security First: Always choose Managed Identities and VNETs over keys when given the option. 安全第一： 有选择时，总是优先选择托管标识和 VNET，而不是密钥。 セキュリティ第一： 選択肢がある場合は、キーよりもマネージド ID と VNET を常に選択してください。
  • Containers: Know when to use disconnected containers for high-security environments. 容器： 知道何时在高安全性环境中使用断开连接的容器。 コンテナ： 高セキュリティ環境で切断されたコンテナをいつ使用するかを知っておいてください。

Latest Changes (2024-2025)

最新变化 (2024-2025)

最新の変更 (2024-2025)

The exam now heavily emphasizes Generative AI integration. You must know how traditional NLP (like CLU) interacts with LLMs (Azure OpenAI).

现在的考试非常强调 生成式 AI 的集成。您必须了解传统 NLP（如 CLU）如何与 LLMs (Azure OpenAI) 交互。

現在の試験では、生成 AI の統合が非常に重視されています。従来の NLP（CLU など）が LLM（Azure OpenAI）とどのように相互作用するかを知っておく必要があります。
